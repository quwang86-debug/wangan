# 网安线上通知书 H5

Vue 3 + Vite 移动端流程应用（欢迎 → 核验 → 通知书 → 合影 → 注册）。

## 本地开发

```bash
pnpm install
pnpm dev
```

访问 http://localhost:5173

## 构建

```bash
pnpm build
pnpm preview
```

## 在线演示（GitHub Pages）

- 仓库：https://github.com/quwang86-debug/wangan
- 访问：https://quwang86-debug.github.io/wangan/

推送 `main` 分支后 GitHub Actions 自动构建并部署。
